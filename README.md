## Installation

1. **Install dependencies:**
```bash
npm install
```

2. **Start development server:**
```bash
npm run dev
```

Or use:
```bash
npm start
```

The application will open at `http://localhost:3000` (or `http://localhost:5173` depending on your setup).

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build/` directory.
